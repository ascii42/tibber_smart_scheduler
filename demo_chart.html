Tibber Smart Scheduler - Chart Visualization Demo

Chart Features:

  • Price Line: Shows electricity prices over 24 hours
  • Scheduled Window: Orange bars show optimal charging times
  • Current Time: Red dashed line shows current time
  • Grid: Makes it easy to read exact prices and times

Live Chart Preview

How it works in Home Assistant:

  • Chart data is available as price_chart_url attribute
  • Raw data available as price_chart_data attribute
  • Schedule windows in schedule_windows attribute
  • Can be displayed in dashboard cards or custom Lovelace cards

Usage in Dashboard:

type: picture
image: "{{ states.switch.tibber_scheduler_dishwasher.attributes.price_chart_url }}"
title: "Dishwasher Schedule"
